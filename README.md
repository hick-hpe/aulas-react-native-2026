# App Biscoito da Sorte

Projeto simples sobre um biscoito da sorte. Quando o usuário abre o biscoito, é exibido uma mensagem.

## Instalar
- Clone o repositório:
    ```bash
    git clone -b atv02 https://github.com/hick-hpe/aulas-react-native-2026/
    ```

- Instale as dependências:
    ```bash
    npm install
    ```

## Executar
Inicie servidor do `Expo`:
```bash
npx expo start
```
